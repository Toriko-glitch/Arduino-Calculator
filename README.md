# Arduino-Calculator
PBL - Arduino Calculator for Intro to IoT


## Course
Introduction to IoT for Electronics

## Description
A basic four-function calculator built with an Arduino Uno. The user enters numbers using a 4x4 membrane keypad and sees the expression and result on a 16x2 LCD display. Supports addition, subtraction, multiplication, and division with division-by-zero error handling.

## Components
- Arduino Uno R3
- 4x4 Membrane Keypad (input)
- 16x2 LCD Display HD44780 (output)
- 1kΩ Resistor (LCD contrast)

## Libraries
- LiquidCrystal.h (standard - LCD control)
- Keypad.h (external - keypad reading)

## Keypad Mapping
- Digits: 0-9
- A = Addition (+)
- B = Subtraction (-)
- C = Multiplication (*)
- D = Division (/)
- '#' = Equals
- '*' = Clear
